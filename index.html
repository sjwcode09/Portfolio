02

Skils

UX/UI 디자인, Frontend 및 Backend 개발, DB 관리, SCM

{ UI/ UX }

HTML, CSS, JavaScript로 UX/UI를 개발하고, Figma를 활용하여 디자인을 구상하며, SCSS를 사용하여 스타일링을 처리하는 디자인 및 개발 환경입니다.

{ FrontEnd }

React와 Redux를 사용하여 사용자 인터페이스를 개발합니다. JSX를 통해 컴포넌트를 작성하고, React Router를 사용하여 다양한 페이지 간의 라우팅을 관리합니다. Axios를 활용하여 서버와의 HTTP 통신을 수행하며, 서버로부터 데이터를 비동기적으로 요청하고 받아옵니다.

{ BackEnd }

Node.js와 Express를 사용하여 서버를 구축합니다. 이 서버는 클라이언트로부터의 요청을 받아들이고, 데이터베이스와의 상호작용을 관리하며, AWS S3와의 통합을 통해 파일 업로드 및 다운로드를 처리합니다.

{ DB }

MongoDB는 문서 지향 NoSQL 데이터베이스로, 유연하고 확장 가능한 데이터 저장 및 검색 솔루션을 제공합니다.

{ SCM }

Git과 GitHub을 사용하여 소스 코드를 버전 관리하고 협업하는 환경입니다.
